## [1.0.0] - 2024-12-15

- ✨ 本地搜索功能，支持中文搜索
- ✨ 自定义 Hero 组件和动画效果
- ✨ GitHub Actions 自动化部署
- ✨ 多种背景和 Logo 样式选择
- ✨ Ubuntu Mono 字体的代码块样式
- ✨ 渐变橘黄色主题色彩

## 版本说明

### 版本号规则

- **主版本号**: 不兼容的 API 修改
- **次版本号**: 向下兼容的功能性新增
- **修订号**: 向下兼容的问题修正

### 变更类型

- `新增` - 新功能
- `修改` - 对现有功能的变更
- `弃用` - 即将移除的功能
- `移除` - 已移除的功能
- `修复` - 问题修复
- `安全` - 安全相关修复

### 图标说明

- ✨ 新功能
- 🐛 Bug 修复
- ⚡ 性能优化
- 🎨 样式/UI 改进
- 📝 文档更新
- 🔧 配置修改
- 🚀 部署相关
- 🛠️ 开发工具
- ⚙️ 配置文件
- 📱 移动端适配
- 🌐 国际化
- 🔒 安全修复
